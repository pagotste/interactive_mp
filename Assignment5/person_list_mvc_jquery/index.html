What you can do with jQuery?

jQuery can be used to handle Document Object Model (DOM) elements (finding, selecting, altering), for example selecting all elements with the same property, altering its style/attributes, adding event hanldler to this element and so on. You can also parse JSON files, do AJAX, create effects and animations, or even add plugins.

How it can help you as web developer?

jQuery simplifies the whole process for this DOM elements handling, by being clear and brief, so that web developers work more efficiently. Since it eliminates cross-browser incompatibilities, it is also easier for web devs to develop cross platform web apps. jQuery is also highly extensible thanks to plugins, so web devs can share plugins or use ready-made ones.

If you think our earlier participant list app, in what kind of issues you could use jQuery?

Change html with jQuery instead of using the .innerHTML method or using .append to add the new names, in other words, everything related to DOM manipulation.

What are CDNs? What kind of benefits you get if you use them?

Content delivery network (CDN) is a computer network that is used to share content or data to users through the Internet, based on the user's locaisation. Using CDNs speed up the content delivery in cases of high taffic websites. They also make te delivery faster thanks to several CDN servers existing most of the time, so that the user gets to download content from the closest server. CNDs also reduce bandwidth consumption and also secure data.

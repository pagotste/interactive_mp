Definitions

HTML

HTML (which stands for Hypertext Markup Language) is a language used to create web pages.
It creates hypertext thanks to markups, which is why it is called like that. HTML's markups are used to format the content : text, links, images, animations ... You need a web browser to display HTML pages.

CSS

CSS (Cascading Style Sheets) is a language used to manage the design of HTML pages.
Thanks to CSS, you can set styles of every markup on the HTML page, instead of setting it individually. You can load CSS files directly through HTML code.

JavaScript

JavaScript is a language used on web pages. It is a client based language : the client (user's web browser) will execute the code (opposed to server based languages).
JavaScript creates small scripts on HTML pages in order to execute an effect, or a small animation. Most of the time JS is used to enhance the user interaction with the web page.
One of JavaScript's advantages is that the user doesn't need to reload the page to execute the JS code.

HTTP

HTTP (Hypertext Transfer Protocol) is a protocol used to establish communication between a client (web browser for example) and a web server.
The client sends a request (HTML page,CSS..) to the server, the server looks for the information, interprets the result before sending a reply to the client.

HTML5

HTML5 is the latest major revision of HTML, released in 2014. It is usually used alongside with CSS3 and JavaScript.
HTML5 added and updated previous HTML revisions specifications (such as markups), and introduced a need for a "doctype" declaration at the beginning of HTML pages. The encoding process has also been updated.

Dynamic HTML

Dynamic HTML (DHTML) is the name given to a set of techniques used to make a web page dynamic by modifying itself without reloading the page. It involves mostly event handling and real-time positioning thanks to the use of CSS, JavaScript, HTML and the DOM.
